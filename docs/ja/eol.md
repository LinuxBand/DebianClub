# Debian リリースのサポート終了（EOL）

使用中の Debian バージョンのライフサイクルを把握することは、システムが継続してセキュリティアップデートと技術サポートを受け続けるために非常に重要です。

## Debian のリリースモデル

Debian には主に 3 つのリリースブランチがあります：

-   **Stable（安定版）**: これが公式リリースであり、本番環境での使用が推奨されます。広範なテストを経ており、最高の安定性とセキュリティを提供します。
-   **Testing（テスト版）**: このブランチには、次の安定版リリースに向けたパッケージが含まれています。比較的安定していますが、`stable` ほどの信頼性はありません。新しいソフトウェアを求める開発者や上級ユーザーに適しています。
-   **Unstable（不安定版・コードネーム Sid）**: これは Debian パッケージの活発な開発ブランチです。最新のソフトウェアが含まれていますが、安定性の問題が生じる可能性があり、ミッションクリティカルなシステムには推奨されません。

## 長期サポート（LTS）

公式リリース後、各 Debian 安定版は Debian セキュリティチームから約 3 年間のセキュリティサポートを受けます。

この期間終了後、ボランティアと企業からなる独立したチームが引き継ぎ、追加の長期サポート（LTS）を提供します（通常さらに 2 年間）。これにより、各 Debian 安定版リリースは合計約 5 年間のセキュリティアップデートを受けられます。

::: tip
`〜` が付いた日付は推定値であり、若干変更される可能性があります。最新の正確な情報については、Debian の公式アナウンスをご確認ください。
:::

現在使用中のバージョンの LTS 期間が終了する前に、新しい安定版へのアップグレードを計画・実施することを強くお勧めします。これにより、システムのセキュリティが維持されます。
