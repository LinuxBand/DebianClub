# O que há de novo no Debian 13 (Trixie)

O Debian 13 com o codinome "Trixie" foi lançado oficialmente em agosto de 2025 e é a versão estável atual. A partir de março de 2026, o último lançamento pontual é o **13.4**.

## Principais novos recursos

### Atualizações de pacotes de software

| Software | Versão Debian 13 | Versão Debian 12 | Melhorias chave |
|----------|-----------------|-----------------|-----------------|
| **Kernel Linux** | 6.12 | 6.1 | Melhor suporte a hardware, otimização de desempenho |
| **Python** | 3.13 | 3.11 | Execução mais rápida, novos recursos de linguagem |
| **GCC** | 14.2 | 12.2 | Última tecnologia de compilador, suporte a C++23 |
| **GNOME** | 43 | 43 | Interface moderna, experiência do usuário aprimorada |
| **KDE Plasma** | 5.27 | 5.27 | Recursos aprimorados, melhor estabilidade |
| **Firefox ESR** | 115+ | 102 | Privacidade e desempenho aprimorados |

### Suporte a arquiteturas

- **RISC-V 64 bits** (riscv64) — Primeiro suporte oficial
- **amd64**, **arm64**, **armhf**, **ppc64el**, **s390x** continuam suportados

### Mudanças nos repositórios

```bash
# Nova estrutura de componentes
main contrib non-free-firmware

# Substitui o anterior
main contrib non-free
```
