# Debian 13 (Trixie)の新機能

コードネーム「Trixie」のDebian 13は2025年8月に正式リリースされ、現在の安定版リリースです。2026年3月時点での最新ポイントリリースは**13.4**です。

## 主な新機能

### ソフトウェアパッケージの更新

| ソフトウェア | Debian 13バージョン | Debian 12バージョン | 主な改善点 |
|-----------|------------------|------------------|----------|
| **Linuxカーネル** | 6.12 | 6.1 | より優れたハードウェアサポート、パフォーマンス最適化 |
| **Python** | 3.13 | 3.11 | 高速な実行速度、新しい言語機能 |
| **GCC** | 14.2 | 12.2 | 最新コンパイラ技術、C++23サポート |
| **GNOME** | 43 | 43 | モダンなインターフェース、改善されたユーザー体験 |
| **KDE Plasma** | 5.27 | 5.27 | 拡張機能、安定性の向上 |
| **Firefox ESR** | 115+ | 102 | 強化されたプライバシー保護とパフォーマンス |

### アーキテクチャサポート

- **RISC-V 64ビット** (riscv64) — 初の公式サポート
- **amd64**, **arm64**, **armhf**, **ppc64el**, **s390x** 継続サポート

### リポジトリの変更

```bash
# 新しいコンポーネント構造
main contrib non-free-firmware

# 以前の構造を置き換え
main contrib non-free
```
