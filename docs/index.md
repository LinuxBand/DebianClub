## 💡 为什么选择 Debian？

::: tip 稳定可靠
Debian 以稳定著称，是服务器与关键系统的首选。
:::

::: info 完全免费
完全自由软件，永远免费开源，可自由使用与分发。
:::

::: warning 安全第一
专业安全团队维护，及时发布安全更新与补丁。
