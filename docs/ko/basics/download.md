# Debian 13 다운로드

::: tip 인터랙티브 다운로드
[인터랙티브 다운로드 페이지](/ko/download)에서 지역 및 미러를 선택하세요.
:::

## 다운로드 선택

| 사용 사례 | 권장 다운로드 | 크기 |
|---------|------------|------|
| **대부분의 사용자** | DVD-1 (전체 데스크톱) | ~3.7GB |
| **최소 설치** | netinst (네트워크 설치) | ~400MB |
| **오프라인 설치** | 전체 DVD 세트 | ~20GB |

## 공식 다운로드 소스

- **메인 사이트**: [debian.org/download](https://www.debian.org/download)
- **CD 이미지**: [cdimage.debian.org](https://cdimage.debian.org/)
- **미러 목록**: [debian.org/mirror/list](https://www.debian.org/mirror/list)

## 다운로드 확인

```bash
# SHA256으로 파일 무결성 확인
sha256sum debian-13.4.0-amd64-netinst.iso
```
