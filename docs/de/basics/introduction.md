# Einführung in Debian 13

Willkommen in der Welt von Debian! Wenn Sie neu bei Linux sind oder mehr über die neuen Funktionen von Debian 13 erfahren möchten, bietet Ihnen dieser Artikel eine umfassende Einführung.

## Was ist Debian?

Debian ist ein vollständig von Freiwilligen gepflegtes freies Betriebssystem. Es ist weltweit bekannt für seine Stabilität, Sicherheit und sein leistungsstarkes Paketverwaltungs­system.

::: tip 💡 Tipp für Einsteiger
Linux ist ein Betriebssystem ähnlich wie Windows oder macOS, aber vollständig kostenlos und open-source. Debian ist eine von vielen Linux-„Distributionen" (verschiedene Versionen).
:::

### Die Kernphilosophie von Debian

- **Freie Software**: Alle Software ist open-source; Sie können sie frei nutzen, verändern und weitergeben
- **Stabilität an erster Stelle**: Neue Funktionen werden erst nach gründlicher Prüfung aufgenommen
- **Community-gesteuert**: Gepflegt und entwickelt von Freiwilligen weltweit

## Neue Funktionen in Debian 13

Debian 13 (Codename: Trixie) bringt viele aufregende neue Funktionen:

### 🚀 System­verbesserungen

- **Linux-Kernel 6.12**: Bessere Hardware­unterstützung und Leistungs­optimierung
- **RISC-V-Architektur­unterstützung**: Offizielle Unterstützung für 64-Bit-RISC-V (riscv64)-Architektur
- **ARM64-Sicherheits­verbesserungen**: Unterstützung für PAC/BTI-Sicherheits­funktionen
- **Verbesserter Installer**: Benutzer­freundlichere Installations­erfahrung
- **HTTP-Boot-Unterstützung**: Unterstützung für UEFI HTTP Boot

### 🖥️ Desktop-Umgebungs­updates

- **GNOME 43**: Moderne Benutzer­oberfläche mit verbesserter Benutzer­erfahrung
- **KDE Plasma 5.27**: Funktions­reiche Desktop-Umgebung
- **Xfce 4.18**: Leichtgewichtig, aber voll funktions­fähig
- **MATE 1.26**: Traditionelle Desktop-Erfahrung
- **LXQt 1.2.0**: Leichtgewichtige Qt-Desktop-Umgebung

### 📦 Software­paket-Updates

| Software | Version | Wichtige Verbesserungen |
|----------|---------|-------------------------|
| Linux-Kernel | 6.12 | Bessere Hardware­unterstützung und Leistung |
| LibreOffice | 7.4+ | Verbesserte Dokument­kompatibilität |
| Python | 3.13 | Schnellere Ausführung und neue Funktionen |
| GCC | 14.2 | Neueste Compiler­unterstützung |

## Warum Debian wählen?

### ✅ Vorteile

::: tip 🔒 Überragende Stabilität
Debians „Stable"-Release wird über einen langen Zeitraum ausgiebig getestet und weist selten System­abstürze oder Software­probleme auf.
:::

::: tip 🛡️ Sicher und zuverlässig
Es gibt ein dediziertes Sicherheits­team, das Sicherheits­lücken umgehend behebt.
:::

::: tip 📦 Umfangreiches Software-Repository
Offizielle Repositories enthalten über 50.000 Software­pakete, die nahezu alle Bedürfnisse abdecken.
:::

::: tip 💰 Vollständig kostenlos
Keine Gebühren erforderlich, keine Werbung und keine Tracking-Software.
:::

### ⚠️ Zu beachtende Punkte

::: warning Lern­kurve
Wenn Sie neu bei Linux sind, kann die Eingewöhnung etwas Zeit in Anspruch nehmen.
:::

::: warning Software­versionen
Zur Gewährleistung der Stabilität ist manche Software möglicherweise nicht in der neuesten Version verfügbar.
:::

| Merkmal | Debian | Windows |
|---------|--------|---------|

## Systemanforderungen

Bevor Sie Debian 13 installieren, stellen Sie sicher, dass Ihr Computer die folgenden Anforderungen erfüllt:

### 🖥️ Mindest­konfiguration

- **Prozessor**: 1 GHz oder schneller
- **Arbeitsspeicher**: 2 GB RAM (4 GB empfohlen)
- **Speicher**: 25 GB verfügbarer Festplatten­speicher
- **Grafik**: Unterstützung für 1024×768-Auflösung

### 💪 Empfohlene Konfiguration

- **Prozessor**: Dual-Core 2 GHz oder schneller
- **Arbeitsspeicher**: 8 GB RAM oder mehr
- **Speicher**: 50 GB SSD-Laufwerk
- **Grafik**: Hardware­beschleunigte dedizierte Grafikkarte

::: warning ⚠️ Wichtiger Hinweis
Sichern Sie vor der Installation unbedingt Ihre wichtigen Daten! Obwohl Sie Ihr vorhandenes System für eine Dual-Boot-Installation behalten können, ist ein Backup immer eine kluge Entscheidung.
:::
