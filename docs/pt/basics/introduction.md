# Introdução ao Debian 13

Bem-vindo ao mundo do Debian! Se você é novo no Linux ou quer conhecer as novas funcionalidades do Debian 13, este artigo fornecerá uma introdução abrangente.

## O que é Debian?

Debian é um sistema operacional livre mantido inteiramente por voluntários. É reconhecido mundialmente por sua estabilidade, segurança e poderoso sistema de gerenciamento de pacotes.

::: tip 💡 Dica para iniciantes
Linux é um sistema operacional semelhante ao Windows ou macOS, mas completamente gratuito e de código aberto. Debian é uma das muitas "distribuições" Linux (diferentes versões).
:::

### A filosofia central do Debian

- **Software livre**: Todo software é open-source, você pode usar, modificar e distribuir livremente
- **Estabilidade primeiro**: Novas funcionalidades só são adicionadas após testes completos
- **Orientado pela comunidade**: Mantido e desenvolvido por voluntários de todo o mundo

## Novas funcionalidades do Debian 13

Debian 13 (codinome: Trixie) traz muitas novas e empolgantes funcionalidades:

### 🚀 Melhorias do sistema

- **Kernel Linux 6.12**: Melhor suporte de hardware e otimização de desempenho
- **Suporte à arquitetura RISC-V**: Suporte oficial para 64 bits RISC-V (riscv64)
- **Aprimoramentos de segurança ARM64**: Suporte para recursos de segurança PAC/BTI
- **Instalador aprimorado**: Experiência de instalação mais amigável
- **Suporte HTTP Boot**: Suporte a UEFI HTTP Boot

### 🖥️ Atualizações de ambientes de desktop

- **GNOME 48**: Interface moderna com experiência de usuário aprimorada
- **KDE Plasma 6.3**: Ambiente de desktop rico em recursos
- **Xfce 4.18**: Leve mas completo
- **MATE 1.26**: Experiência de desktop tradicional
- **LXQt 1.2.0**: Ambiente de desktop Qt leve

### 📦 Atualizações de pacotes de software

| Software | Versão | Principais melhorias |
|----------|--------|---------------------|
| Kernel Linux | 6.12 | Melhor suporte de hardware e desempenho |
| LibreOffice | 25.2 | Compatibilidade de documentos aprimorada |
| Python | 3.12 | Velocidade de execução mais rápida e novos recursos |
| GCC | 15.x | Suporte ao compilador mais recente |
| GNOME | 48 | Ambiente de desktop moderno |
| KDE Plasma | 6.3 | Experiência de desktop rica em recursos |

## Por que escolher Debian?

### ✅ Vantagens

::: tip 🔒 Estabilidade excepcional
A versão "stable" do Debian é testada por muito tempo, raramente ocorrem travamentos do sistema ou problemas de software.
:::

::: tip 🛡️ Seguro e confiável
Uma equipe de segurança dedicada corrige vulnerabilidades rapidamente.
:::

::: tip 📦 Software abundante
O repositório oficial contém mais de 50.000 pacotes, cobrindo quase todas as necessidades.
:::

::: tip 💰 Completamente gratuito
Sem custos, sem anúncios, sem software de rastreamento.
:::
