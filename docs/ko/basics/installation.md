# Debian 13 설치 가이드

## 설치 전 준비

- ✅ 부팅 가능한 미디어 생성 완료
- ✅ 중요 데이터 백업 완료
- ✅ 네트워크 연결 사용 가능 (권장)

::: warning 중요
새로 설치하면 하드 디스크 데이터가 **완전히 삭제**됩니다! 중요한 파일을 백업하세요.
:::

## 설치 프로그램 시작

### BIOS/UEFI 설정 접근

일반적인 키:
- **Dell**: F2 또는 F12
- **HP**: F10 또는 F9
- **Lenovo**: F1 또는 F12
- **ASUS**: F2 또는 Del

### 설치 단계

1. **언어 선택** — 한국어 또는 선호 언어
2. **네트워크 구성** — 유선 연결 권장
3. **디스크 파티션** — 초보자는 가이드 파티션 사용
4. **기본 시스템 설치** — 자동 다운로드
5. **소프트웨어 선택** — 데스크톱 환경 선택
6. **GRUB 부트로더 설치** — 메인 디스크에 설치
7. **설치 완료** — 재시작

## 권장 파티션 레이아웃

| 파티션 | 크기 | 파일 시스템 | 비고 |
|-------|-----|-----------|------|
| `/boot/efi` | 512MB | FAT32 | UEFI 시스템만 |
| `/` | 20-50GB | ext4 | 루트 파티션 |
| `swap` | 2-8GB | swap | 가상 메모리 |
| `/home` | 나머지 공간 | ext4 | 사용자 데이터 |
