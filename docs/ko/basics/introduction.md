# Debian 13 소개

Debian의 세계에 오신 것을 환영합니다! Linux를 처음 접하거나 Debian 13의 새로운 기능에 대해 알고 싶다면, 이 글에서 포괄적인 소개를 드리겠습니다.

## Debian이란?

Debian은 완전히 자원봉사자들이 유지관리하는 자유 운영체제입니다. 안정성, 보안성, 강력한 패키지 관리 시스템으로 전 세계적으로 유명합니다.

::: tip 💡 초보자 팁
Linux는 Windows나 macOS와 유사한 운영체제이지만, 완전히 무료이고 오픈소스입니다. Debian은 수많은 Linux "배포판"(다른 버전) 중 하나입니다.
:::

### Debian의 핵심 철학

- **자유 소프트웨어**: 모든 소프트웨어가 오픈소스이며, 자유롭게 사용, 수정, 배포 가능
- **안정성 우선**: 철저한 테스트를 거친 후에만 새 기능 추가
- **커뮤니티 주도**: 전 세계 자원봉사자들이 유지관리 및 개발

## Debian 13의 새로운 기능

Debian 13(코드명: Trixie)은 많은 흥미로운 새 기능을 제공합니다:

### 🚀 시스템 개선

- **Linux 커널 6.12**: 더 나은 하드웨어 지원 및 성능 최적화
- **RISC-V 아키텍처 지원**: 64비트 RISC-V(riscv64) 아키텍처 공식 지원
- **ARM64 보안 강화**: PAC/BTI 보안 기능 지원
- **개선된 설치 프로그램**: 더 사용자 친화적인 설치 경험
- **HTTP 부트 지원**: UEFI HTTP Boot 지원

### 🖥️ 데스크톱 환경 업데이트

- **GNOME 48**: 현대적인 UI와 향상된 사용자 경험
- **KDE Plasma 6.3**: 기능이 풍부한 데스크톱 환경
- **Xfce 4.18**: 가볍지만 기능 완전
- **MATE 1.26**: 전통적인 데스크톱 경험
- **LXQt 1.2.0**: 가벼운 Qt 데스크톱 환경

### 📦 소프트웨어 패키지 업데이트

| 소프트웨어 | 버전 | 주요 개선사항 |
|-----------|------|-------------|
| Linux 커널 | 6.12 | 더 나은 하드웨어 지원 및 성능 |
| LibreOffice | 25.2 | 향상된 문서 호환성 |
| Python | 3.12 | 더 빠른 실행 속도 및 새 기능 |
| GCC | 15.x | 최신 컴파일러 지원 |
| GNOME | 48 | 현대적인 데스크톱 환경 |
| KDE Plasma | 6.3 | 기능이 풍부한 데스크톱 경험 |

## Debian을 선택하는 이유

### ✅ 장점

::: tip 🔒 뛰어난 안정성
Debian의 "stable" 버전은 오랜 테스트를 거쳐, 시스템 충돌이나 소프트웨어 문제가 거의 발생하지 않습니다.
:::

::: tip 🛡️ 안전하고 신뢰할 수 있음
전담 보안팀이 있어 보안 취약점을 신속히 수정합니다.
:::

::: tip 📦 풍부한 소프트웨어
공식 저장소에 50,000개 이상의 패키지가 있어 거의 모든 요구사항을 충족합니다.
:::

::: tip 💰 완전 무료
어떠한 비용도 없고 광고나 추적 소프트웨어도 없습니다.
:::
