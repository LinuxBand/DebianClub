# Baixar Debian 13

::: tip Download interativo
Visite a [página de download interativo](/pt/download) para selecionar sua região e espelho.
:::

## Seleção de download

| Caso de uso | Download recomendado | Tamanho |
|------------|---------------------|---------|
| **A maioria dos usuários** | DVD-1 (Desktop completo) | ~3,7 GB |
| **Instalação mínima** | netinst (Instalação em rede) | ~400 MB |
| **Instalação offline** | Conjunto completo de DVD | ~20 GB |

## Fontes de download oficiais

- **Site principal**: [debian.org/download](https://www.debian.org/download)
- **Imagens CD**: [cdimage.debian.org](https://cdimage.debian.org/)
- **Lista de espelhos**: [debian.org/mirror/list](https://www.debian.org/mirror/list)

## Verificar seu download

```bash
# Verificar integridade do arquivo com SHA256
sha256sum debian-13.4.0-amd64-netinst.iso
```
