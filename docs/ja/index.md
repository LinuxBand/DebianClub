## 💡 なぜ Debian を選ぶのか？

::: tip 🔒 安定性と信頼性
Debian はその優れた安定性で世界的に知られており、多くのサーバーや重要システムに選ばれています
:::

::: tip 🆓 完全無料
Debian は真に自由なソフトウェアであり、永続的に無料かつオープンソースです
:::

::: tip 🛡️ セキュリティ最優先
強力なセキュリティチームがシステムのセキュリティアップデートを適時リリースします
:::

::: tip 📦 豊富なソフトウェア
50,000 以上のプリコンパイル済みソフトウェアパッケージが利用可能で、新たに追加された RISC-V 64 ビットアーキテクチャを含む複数のアーキテクチャをサポート
:::
