# Debian 13インストールガイド

## インストール前の準備

- ✅ ブータブルメディアが作成済み
- ✅ 重要なデータがバックアップ済み
- ✅ ネットワーク接続が利用可能（推奨）

::: warning 重要
フレッシュインストールはハードディスクのデータを**完全に消去**します！重要なファイルをバックアップしてください。
:::

## インストーラーの起動

### BIOS/UEFIへのアクセス

一般的なキー：
- **Dell** : F2またはF12
- **HP** : F10またはF9
- **Lenovo** : F1またはF12
- **ASUS** : F2またはDel

### インストール手順

1. **言語の選択** — 日本語またはお好みの言語
2. **ネットワークの設定** — 有線接続推奨
3. **ディスクのパーティション** — 初心者にはガイド付きパーティション
4. **ベースシステムのインストール** — 自動ダウンロード
5. **ソフトウェアの選択** — デスクトップ環境を選択
6. **GRUBブートローダーのインストール** — メインディスクにインストール
7. **インストールの完了** — 再起動

## 推奨パーティション構成

| パーティション | サイズ | ファイルシステム | 備考 |
|------------|------|--------------|------|
| `/boot/efi` | 512MB | FAT32 | UEFIシステムのみ |
| `/` | 20-50GB | ext4 | ルートパーティション |
| `swap` | 2-8GB | swap | 仮想メモリ |
| `/home` | 残りの容量 | ext4 | ユーザーデータ |
