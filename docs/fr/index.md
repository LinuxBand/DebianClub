## 💡 Pourquoi choisir Debian ?

::: tip 🔒 Stable et fiable
Debian est réputée pour son excellente stabilité, le choix privilégié pour de nombreux serveurs et systèmes critiques
:::

::: tip 🆓 Entièrement gratuit
Debian est un vrai logiciel libre, gratuit et open-source pour toujours
:::

::: tip 🛡️ La sécurité avant tout
Une équipe de sécurité robuste garantit la publication rapide des mises à jour de sécurité système
:::

::: tip 📦 Logiciels abondants
Plus de 50 000 paquets logiciels précompilés disponibles, avec support pour plusieurs architectures dont la nouvelle architecture RISC-V 64 bits
:::
