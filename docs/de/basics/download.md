# Debian 13 herunterladen

::: tip Interaktiver Download
Besuchen Sie die [interaktive Download-Seite](/de/download), um Ihre Region und einen Spiegel auszuwählen.
:::

## Download-Auswahl

| Anwendungsfall | Empfohlener Download | Größe |
|---------------|---------------------|-------|
| **Die meisten Nutzer** | DVD-1 (Vollständiger Desktop) | ~3,7 GB |
| **Minimale Installation** | netinst (Netzwerk-Installation) | ~400 MB |
| **Offline-Installation** | Vollständiges DVD-Set | ~20 GB |

## Offizielle Download-Quellen

- **Hauptseite**: [debian.org/download](https://www.debian.org/download)
- **CD-Images**: [cdimage.debian.org](https://cdimage.debian.org/)
- **Spiegelliste**: [debian.org/mirror/list](https://www.debian.org/mirror/list)

## Download verifizieren

```bash
# Dateiintegrität mit SHA256 prüfen
sha256sum debian-13.4.0-amd64-netinst.iso
```
