## 💡 Por que escolher o Debian?

::: tip 🔒 Estável e confiável
O Debian é reconhecido pela sua excelente estabilidade, sendo a escolha preferida de muitos servidores e sistemas críticos
:::

::: tip 🆓 Completamente gratuito
O Debian é software verdadeiramente livre, gratuito e de código aberto para sempre
:::

::: tip 🛡️ Segurança em primeiro lugar
Uma poderosa equipe de segurança garante a publicação oportuna de atualizações de segurança do sistema
:::

::: tip 📦 Software abundante
Mais de 50.000 pacotes de software pré-compilados disponíveis, com suporte para múltiplas arquiteturas, incluindo a nova arquitetura RISC-V de 64 bits
:::
