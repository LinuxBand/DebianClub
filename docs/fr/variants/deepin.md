# Deepin

Deepin est une distribution Linux développée et maintenue par Wuhan Deepin Technology Co., Ltd. en Chine. Basée sur Debian stable, elle est connue pour son DDE (Deepin Desktop Environment), considéré comme l'un des environnements de bureau les plus visuellement soignés sur n'importe quel système d'exploitation.

## Informations de base

| Attribut | Détails |
|---|---|
| Basé sur | Debian stable |
| Cycle de publication | Sorties régulières suivant Debian stable |
| Bureau par défaut | DDE (Deepin Desktop Environment) |
| Utilisateurs cibles | Utilisateurs chinois et mondiaux cherchant un beau bureau |
| Site web | [https://www.deepin.org](https://www.deepin.org) |

## Fonctionnalités clés

- **DDE visuellement époustouflant** : Interface fluide et moderne avec animations élégantes, dock inspiré de macOS et Centre de Contrôle intuitif.
- **Deepin App Store** : Magasin d'applications graphique avec de nombreuses applications Linux populaires plus faciles à installer.
- **Deepin Wine** : Couche de compatibilité intégrée permettant à de nombreuses applications Windows populaires de fonctionner sur Deepin.
- **Applications Deepin originales** : Suite d'outils propre incluant Deepin Music, Deepin Movie, Deepin Terminal.
- **Profondément localisé** : Excellente prise en charge de la langue chinoise et des services locaux pour les utilisateurs en Chine.

## Pour commencer

```bash
# Télécharger depuis : https://www.deepin.org/en/download/

# Après installation, mettre à jour le système
sudo apt update && sudo apt upgrade -y

# Activer l'optimisation des performances
sudo systemctl enable --now deepin-turbo

# Installer des applications supplémentaires via le Centre d'applications
# (interface graphique disponible dans le dock)
```
