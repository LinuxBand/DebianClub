# Debian 릴리스 지원 종료(EOL)

사용 중인 Debian 버전의 수명 주기를 파악하는 것은 시스템이 보안 업데이트와 기술 지원을 계속 받을 수 있도록 하는 데 매우 중요합니다.

## Debian 릴리스 모델

Debian에는 주로 세 가지 릴리스 브랜치가 있습니다:

-   **Stable(안정)**: 공식 릴리스이며 프로덕션 환경에서의 사용을 권장합니다. 광범위한 테스트를 거쳤으며 최고의 안정성과 보안을 제공합니다.
-   **Testing(테스트)**: 이 브랜치에는 다음 안정 릴리스에 포함될 패키지가 들어 있습니다. 비교적 안정적이지만 `stable` 버전만큼 신뢰성이 높지는 않습니다. 더 새로운 소프트웨어를 원하는 개발자와 고급 사용자에게 적합합니다.
-   **Unstable(불안정, 코드명 Sid)**: Debian 패키지의 활발한 개발 브랜치입니다. 최신 소프트웨어를 포함하지만 안정성 문제가 있을 수 있으며 미션 크리티컬 시스템에는 권장되지 않습니다.

## 장기 지원(LTS)

공식 릴리스 후, 각 Debian 안정 버전은 Debian 보안 팀으로부터 약 3년간의 보안 지원을 받습니다.

이 기간이 끝나면 자원봉사자와 기업으로 구성된 독립적인 팀이 인수하여 추가적인 장기 지원(LTS)을 제공합니다(일반적으로 2년 추가). 이로써 각 Debian 안정 릴리스는 총 약 5년간의 보안 업데이트를 받게 됩니다.

아래 표는 최근 Debian 버전의 릴리스 날짜와 지원 종료(EOL) 날짜를 나열합니다.

| 코드명 | 릴리스 날짜 | 일반 지원 종료 | 장기 지원(LTS) 종료 |

| **Debian 11 (Bullseye)** | 2021-08-14   | 2024년 7월             | 2026-06-30                     |
| **Debian 10 (Buster)**   | 2019-07-06   | 2022-09-10             | 2024-06-30                     |
| **Debian 9 (Stretch)**   | 2017-06-17   | 2020-07-06             | 2022-06-30                     |

::: tip
`~`로 표시된 날짜는 추정치이며 약간 변경될 수 있습니다. 가장 정확한 정보는 공식 Debian 공지를 확인하세요.
:::

시스템의 보안을 유지하기 위해, 현재 버전의 LTS 기간이 종료되기 전에 최신 안정 버전으로 업그레이드를 계획하고 실행할 것을 강력히 권장합니다.
