## <i class="ph-duotone ph-lightbulb hi"></i> ¿Por qué elegir Debian?

::: tip 🔒 Estable y confiable
Debian es reconocido por su excelente estabilidad, la opción preferida para muchos servidores y sistemas críticos
:::

::: tip 🆓 Completamente gratuito
Debian es software verdaderamente libre, siempre gratuito y de código abierto
:::

::: tip 🛡️ La seguridad primero
Un potente equipo de seguridad garantiza la publicación oportuna de actualizaciones de seguridad del sistema
:::

::: tip 📦 Software abundante
Más de 50.000 paquetes de software precompilados disponibles, con soporte para múltiples arquitecturas, incluida la nueva arquitectura RISC-V de 64 bits
:::
