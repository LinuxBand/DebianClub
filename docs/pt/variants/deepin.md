# Deepin

Deepin é uma distribuição Linux desenvolvida e mantida pela Wuhan Deepin Technology Co., Ltd. na China. Baseada no Debian stable, é conhecida pelo seu DDE (Deepin Desktop Environment), amplamente considerado um dos ambientes de desktop mais refinados visualmente disponíveis.

## Informações básicas

| Atributo | Detalhes |
|---|---|
| Baseado em | Debian stable |
| Ciclo de lançamento | Lançamentos regulares seguindo o Debian stable |
| Desktop padrão | DDE (Deepin Desktop Environment) |
| Usuários-alvo | Usuários chineses e globais buscando um desktop bonito |
| Site | [https://www.deepin.org](https://www.deepin.org) |

## Recursos principais

- **DDE visualmente impressionante**: Interface fluida e moderna com animações elegantes, dock inspirado no macOS e Centro de Controle intuitivo.
- **Deepin App Store**: Loja de aplicativos gráfica com muitos apps Linux populares mais fáceis de instalar.
- **Deepin Wine**: Camada de compatibilidade integrada que permite que muitos apps Windows populares rodem no Deepin.
- **Aplicativos Deepin originais**: Suite de ferramentas própria incluindo Deepin Music, Deepin Movie, Deepin Terminal.
- **Profundamente localizado**: Excelente suporte ao idioma chinês e serviços locais.

## Primeiros passos
