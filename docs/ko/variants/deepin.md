# Deepin

Deepin은 중국 우한 Deepin Technology Co., Ltd.가 개발 및 유지관리하는 리눅스 배포판입니다. Debian stable 기반으로 자체 개발한 DDE(Deepin Desktop Environment)로 유명하며, 어떤 운영체제에서도 가장 시각적으로 세련된 데스크톱 환경 중 하나로 평가받고 있습니다.

## 기본 정보

| 속성 | 세부 사항 |
|---|---|
| 기반 | Debian stable |
| 릴리스 주기 | Debian stable을 따르는 정기 릴리스 |
| 기본 데스크톱 | DDE (Deepin Desktop Environment) |
| 대상 사용자 | 아름다운 데스크톱을 원하는 중국 및 전 세계 사용자 |
| 웹사이트 | [https://www.deepin.org](https://www.deepin.org) |

## 주요 기능

- **시각적으로 놀라운 DDE**: macOS에서 영감을 받은 독과 직관적인 제어 센터를 갖춘 유연하고 현대적인 인터페이스.
- **Deepin 앱 스토어**: 많은 인기 리눅스 앱을 더 쉽게 설치할 수 있는 그래픽 앱 스토어.
- **Deepin Wine**: 많은 인기 Windows 앱이 Deepin에서 실행될 수 있도록 하는 내장 호환성 레이어.
- **독자적인 Deepin 앱**: Deepin Music, Deepin Movie, Deepin Terminal을 포함한 자체 도구 스위트.
- **깊은 현지화**: 중국 사용자를 위한 우수한 중국어 지원 및 로컬 서비스.

## 시작하기

```bash
# https://www.deepin.org/en/download/ 에서 다운로드

# 설치 후 시스템 업데이트
sudo apt update && sudo apt upgrade -y
```
